## ✨ Features

| Feature | Description |
|---|---|
| 💊 **Medicine Management** | Add conditions with medicines, stock count & custom schedules |
| ⏰ **Smart Reminders** | Local notifications at exact times, independent from system alarm |
| 📉 **Stock Countdown** | Auto-completes when stock runs out — no manual stop needed |
| 🌅 **Morning Check-in** | Daily pop-up to monitor symptoms and overall condition |
| 📋 **History** | View all past conditions and recoveries |
| 🔔 **Custom Notifications** | Style, sound, and vibration preferences |
| 🌐 **Bilingual** | English & Bahasa Indonesia |
| 🔴 **App Badge** | Shows count of pending medicine reminders |

---

## 🔐 Permissions

| Permission | Purpose |
|---|---|
| `POST_NOTIFICATIONS` | Show medicine reminders |
| `SCHEDULE_EXACT_ALARM` | Schedule reminders at exact time |
| `USE_EXACT_ALARM` | Exact alarm support |
| `VIBRATE` | Vibration on reminder |
| `RECEIVE_BOOT_COMPLETED` | Reschedule reminders after reboot |
